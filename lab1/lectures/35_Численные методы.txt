Лекция 3. Решение задачи Коши для дифференциальных уравнений высшего порядка
Отметить как выполненный
Пусть задача Коши имеет вид:
(81)
Решение задачи Коши для одного дифференциального уравнения
n
-го порядка вида (81) сводится к решению задачи Коши для системы из n дифференциальных уравнений первого порядка с помощью замены:
(82)
В результате получим систему дифференциальных уравнений первого порядка вида:
(83)
Начальные условия примут вид:
(84)
Рассмотрим далее решение задачи Коши для системы дифференциальных уравнений первого порядка.
Более детально можно изучить данную лекцию в [4], [5].
4.  Вержбицкий В.М. Численные методы. – М: Высшая школа, 2001. – 382с.
5. Вержбицкий В.М. Основы численных методов. – М.: Высшая школа, 2002. – 840с.
